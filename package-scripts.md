# Available Scripts

## Development

```bash
npm run dev
```
Starts the development server on **port 3004**
- Hot reload enabled
- Access at http://localhost:3004

## Production Build

```bash
npm run build
```
Creates an optimized production build in the `.next` directory

## Start Production Server

```bash
npm start
```
Starts the production server on **port 3004**
- Run `npm run build` first

## Linting

```bash
npm run lint
```
Runs ESLint to check for code issues

## All Commands Summary

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server (port 3004) |
| `npm run build` | Build for production |
| `npm start` | Start production server (port 3004) |
| `npm run lint` | Run linter |
